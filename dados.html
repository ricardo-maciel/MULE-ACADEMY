Shopify Store (JSON)

  "item": "Tênis",
  "preco": 89.90
}
🇪🇸

SAP ERP (XML)

<pedido>
  <item>Tênis</item>
  <preco>89.90</preco>

item,preco
Tênis,89.90

CENA 3: FORMATOS DE DADOS

Dicionário de Idiomas Tech

Passe o mouse ou clique sobre cada idioma para entender suas regras estruturais de forma simples.

🇺🇸

JSON

"O Inglês do Mundo Tech"

É o idioma mais moderno e popular. Organiza as informações como um **dicionário**, emparelhando chaves e valores dentro de chaves `{}`.

{
  "nome": "João",
  "idade": 25
}
Chaves entre aspas duplas Estrutura leve e rápida
🇪🇸

XML

"O Espanhol Estruturado"

Mais formal e tradicional. Funciona como uma **árvore hierárquica** onde cada dado fica guardado entre tags de abertura e fechamento `<>` e `</>`.

<aluno>
  <nome>João</nome>
  <idade>25</idade>
</aluno>
Muito seguro e rígido Perfeito para sistemas legados
🇧🇷

CSV

"O Português Direto"

O mais simples e rústico de todos. É estruturado exatamente como uma **linha de planilha**, separando os valores apenas por vírgulas.

nome,idade
João,25
Computa dados de forma muito compacta Excelente para tabelas grandes
